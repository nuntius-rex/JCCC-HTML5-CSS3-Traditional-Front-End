Flex Responsive Layout

This is a Flex multi-column liquid layout.
Columns can be added by simply droping them in. The term "Responsive" means Liquid+Adaptive. Meaning that the page resizes like a liquid layout automatically by page width but also adapts or "snaps to" when certain sizes are reached.
